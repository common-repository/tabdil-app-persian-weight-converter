This plugin provides a conversion form for all **international and Iranian (Persian) weight units** Anywhere of your site you want.

== Description ==

With this plugin you will easily put a conversion form for converting 41 international and Iranian (Persian) weight units, Anywhere in your website.


= How to use =

1. First **Install** the plugin.
2. Use the plugin's main **Shortcode** anywhere inside any Page or Post: **[tabdil persianWeight]**


== Shortcode ==

Anywhere you put this shortcode **conversion form** whil be shown:
`[tabdil persianWeight]`

== How to edit css ==

The *sass* files are available in the package in the **/sass** folder. You can edit and compile it. You can find the [sass documentation here](https://sass-lang.com/documentation/ "sass documentation").
